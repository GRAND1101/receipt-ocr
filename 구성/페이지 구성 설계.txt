페이지 구성 설계 (보완 버전)

로그인 페이지
경로: /login
기능: Google 계정 로그인, 소개 문구 표시
구성: 구글 로그인 버튼, 앱 소개

영수증 업로드 페이지
경로: /upload
기능: 영수증 이미지 업로드 및 OCR 결과 확인
구성: 이미지 선택 버튼, 미리보기, 결과 테이블

대시보드 페이지
경로: /dashboard
기능: 소비 통계 요약, 월별 통계 차트 시각화
구성: 요약 카드, 원형 차트, 막대 그래프

소비 내역 리스트 페이지
경로: /records
기능: 사용자 소비 내역 리스트 확인
구성: 표 형태 내역, 날짜/카테고리 필터

설정 페이지
경로: /settings
기능: 계정 정보 표시 및 로그아웃
구성: 이메일 정보, 로그아웃 버튼

에러 안내 페이지
경로: /error
기능: OCR 실패 또는 로그인 실패 시 메시지 표시
구성: 에러 메시지, 홈으로 돌아가기 버튼

소개/도움말 페이지
경로: /about
기능: 앱의 기능 및 사용 방법 소개
구성: 간단한 설명 텍스트, 예시 이미지, 안내 문구

404 페이지
경로: *
기능: 존재하지 않는 경로 접근 시 안내
구성: 페이지를 찾을 수 없습니다 메시지, 메인 이동 버튼
